[ 사전 준비 - 설치되어 있어야 하는 것 ]
  - XAMPP (Apache + MySQL)
  - Git
  - Python 3.x
  - Google Drive 데스크톱 앱 (로그인 상태)

========================================================
STEP 1. 코드 받기 (Git Clone)
========================================================

1) XAMPP htdocs 폴더로 이동
   cd C:\xampp\htdocs

2) 프로젝트 클론
   git clone https://github.com/minjunbyeon-netizen/chunking-english.git 03_chunking

3) 클론 확인
   cd 03_chunking
   dir


========================================================
STEP 2. DB 설정 파일 만들기
========================================================

1) config\db.example.php 를 config\db.php 로 복사
   copy config\db.example.php config\db.php

2) config\db.php 파일 열어서 아래 내용 추가
   (기존 DB 설정 아래에 붙여넣기)

   define('APP_URL',        'http://localhost/03_chunking');
   define('RESEND_API_KEY', 're_tyQ1A3ce_FLV56rsEFYQbg5xwanwY2a2u');
   define('MAIL_FROM',      'onboarding@resend.dev');
   define('MAIL_FROM_NAME', '청킹잉글리시');


========================================================
STEP 3. DB 생성 및 데이터 임포트
========================================================

1) XAMPP 실행 → Apache + MySQL Start

2) DB 생성
   "C:\xampp\mysql\bin\mysql.exe" -u root -e "CREATE DATABASE IF NOT EXISTS chunking_english CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"

3) 스키마 임포트
   "C:\xampp\mysql\bin\mysql.exe" -u root chunking_english < database\schema.sql

4) 데이터 임포트 (시간 좀 걸림)
   "C:\xampp\mysql\bin\mysql.exe" -u root --default-character-set=utf8mb4 chunking_english < database\data.sql

5) 테스트용 지자체 추가
   "C:\xampp\mysql\bin\mysql.exe" -u root chunking_english -e "INSERT INTO organizations (name, region, license_code, max_users, is_active) VALUES ('테스트 지자체', '서울', 'TEST0001', 100, 1);"

6) 확인
   "C:\xampp\mysql\bin\mysql.exe" -u root chunking_english -e "SELECT COUNT(*) FROM expressions;"
   → 1050 이 나오면 정상


========================================================
STEP 4. 이미지 / 오디오 파일 동기화
========================================================

Google Drive 데스크톱 앱이 실행 중이어야 함 (G:\내 드라이브 접근 가능)

1) 이미지 복사 (Google Drive → 로컬)
   python -c "import shutil; shutil.copytree(r'G:\내 드라이브\chunking-english\img', r'C:\xampp\htdocs\03_chunking\asset\img', dirs_exist_ok=True); print('이미지 복사 완료')"

2) 오디오 복사 (Google Drive → 로컬)
   python -c "import shutil; shutil.copytree(r'G:\내 드라이브\chunking-english\audio', r'C:\xampp\htdocs\03_chunking\asset\audio', dirs_exist_ok=True); print('오디오 복사 완료')"

※ 이미지 1.6GB라 시간 걸릴 수 있음


========================================================
STEP 5. Python 패키지 설치

pip install requests pymysql gtts elevenlabs


========================================================
STEP 6. 동작 확인
========================================================

브라우저에서 아래 주소 접속:

  관리자 메인:    http://localhost/03_chunking/admin/index.php
  관리자 대시:    http://localhost/03_chunking/admin/dashboard.php

python database\generate_audio_typecast.py --days 17-50 --api-key [Typecast API키]
